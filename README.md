# Omilia Channels API - Sample Project

## Overview

Java example implementation of interacting with DiaManT Channels API.

## Requirements

- JDK (version 1.8.0 or later)

## Installation

### Windows

```bash
mvnw.cmd clean install
```

### Linux and macOS

```bash
./mvnw clean install
```
